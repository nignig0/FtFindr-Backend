# Ft Findr Backend

Ft Findr is a mobile fashion discovery app that helps users find the same or similar clothing items from nearby or local sellers.

This backend supports the app by handling:
- image-based search
- seller and item data
- location-based result prioritization
- search history and bookmarks
- integration with image recognition services such as Google Cloud Vision

## Project Summary

The goal of Ft Findr is to make it easier for people to discover fashion items they like and connect them to local stores, online shops, and social sellers.

The platform focuses on:
- local-first search results
- similar item suggestions when exact matches are unavailable
- a simple mobile-first user experience

## Core Features

- Search using a photo from the camera or gallery
- Find exact or similar fashion items
- Prioritize results based on user location
- Save search history
- Bookmark items for later
- View seller details and external store links

## Architecture

The system is split into two main parts:
- Flutter mobile frontend
- Backend with database and image search/API integration

This repository contains the backend side of the project.
